# 游钓天下 — 收杆中鱼系统 V2

> 参考：Fishing Planet / Ace Fishing / Stardew Valley / Fishing Clash

---

## 一、当前系统问题

| 问题 | 现状 | 影响 |
|------|------|------|
| 咬钩窗口太长 | 5 秒，基本不会失败 | 无紧张感 |
| 搏鱼单调 | 点收线→体力降→结束 | 没有"斗鱼"感觉 |
| 没有张力甜区 | 只有一条红条，满了断线 | 玩家不用思考 |
| QTE 割裂 | 随机弹"松线"，与张力无关 | 烦人，不是决策 |
| 鱼无行为变化 | 纯体力条递减 | 所有鱼一样体验 |

## 二、新系统核心概念

### 张力甜区（Tension Sweet Spot）

```
  0%         20%          60%         85%       100%
  ├───────────┼────────────┼───────────┼──────────┤
  脱钩区      安全区       甜区        警告区     断线
  (太松)     (慢磨)      (最佳伤害)   (危险)    (秒断)

  脱钩区   ：张力 < 20% 超过 3 秒 → 鱼脱钩逃跑
  安全区   ：20-60%，鱼体力下降慢
  甜区     ：60-85%，鱼体力下降最快（1.5 倍伤害）
  警告区   ：85-98%，鱼体力降但随时断
  断线     ：≥ 100% → 瞬间断
```

### 泵吸操作（Pump & Rest）

```
按住收线 → 张力 ↑，鱼体力 ↓
松开     → 张力自然回落

玩家要做的是：让张力在 60-85% 甜区来回摆动
太贪心 → 进警告区 → 断线风险
太保守 → 安全区 → 磨很久，鱼可能脱钩
```

## 三、新的中鱼收杆流程

### Step 1: 咬钩信号增强

| 信号 | 表现 |
|------|------|
| 浮漂 | 剧烈抖动 → 突然箭沉（入水消失） |
| 音效 | 铃铛声 + 溅水声 |
| 震动 | 手机短震 2 次 |
| UI | 屏幕中央闪红光 + "提竿!" |

### Step 2: 提竿窗口（Hook Set）

| 鱼稀有度 | 窗口时间 |
|----------|----------|
| ★ 常见 | 3 秒 |
| ★★ 稀有 | 2.5 秒 |
| ★★★ 史诗 | 2 秒 |
| ★★★★ 神秘 | 1.5 秒 |
| ★★★★★ 传说 | 1 秒 |

未在窗口内提竿 → 鱼脱钩逃跑

### Step 3: 搏鱼（Tension Fight）

```
收线按钮 ─── 按住不放 → 张力持续上升
                   ↓
        张力进入甜区(60-85%) → 鱼体力快速下降
                   ↓
        张力进入警告区(85%+) → 条变红闪烁，危险！
                   ↓
        玩家松手 → 张力回落 → 再按住收线
                   ↓
        鱼挣扎（burst）→ 张力突然暴涨 20-30 点
                   ↓
        鱼体力归零 → 进入抄网/鱼获
```

### Step 4: 鱼挣扎事件（替换 QTE）

| 事件 | 触发 | 表现 | 对策 |
|------|------|------|------|
| 冲刺(burst) | 鱼体力 60%/30% 时 | 张力瞬间 +25，浮漂猛冲 | 立刻松手 |
| 跳跃(jump) | 随机 | 浮漂飞出水面，张力波动 | 不操作，等落水 |
| 深潜(dive) | 鱼体力 40% 时 | 张力缓升，收线效率减半 | 小幅度收线 |

鱼在 tank 体力低位时挣扎更频繁——越到最后越刺激。

## 四、控件调整

```
现在的按钮：
  [松线]  [收线]

改为：
  按住 [收线] → 张力上升，鱼体力下降
  松手 → 张力回落
  点按 [松线] → 快速降张力（紧急泄力）
```

去掉分离的松线/收线按钮。**收线变成按压操作**，不是点击。松线保留为紧急按钮。

或者更简单：**只保留收线按钮**，按压=收线，松手=自然泄力。
