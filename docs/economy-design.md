# 游钓天下 — 经济系统设计

## 一、货币体系

| 货币 | 定位 | 获取方式 | 消耗方式 |
|------|------|----------|----------|
| **金币** | 基础货币，循环核心 | 卖鱼 / 任务 / 签到 / 鱼缸产出 | 买装备 / 修理 / 入场费 / 补签 |
| **钻石** | 高级货币，稀缺 | 签到(少量) / 成就 / 任务 | 稀有道具 / 加速 / 皮肤 |

### 初始数值

| 项目 | 数值 | 说明 |
|------|------|------|
| 初始金币 | **2000** | 能买一套基础装备 + 50 份鱼饵 |
| 初始钻石 | **50** | 体验 1-2 次钻石消费 |
| 初始鱼饵 | **50 份沙蚕** | 够钓 30-40 次（考虑脱钩损失） |

## 二、金币循环

```
        ┌─ 买鱼饵(消耗品) ──┐
        │                   ↓
      修理费 ←── 抛竿钓鱼 ──→ 装备购买(一次性)
        │         ↓
        └── 卖鱼得金币
                  ↓
              鱼缸被动产出
```

### 2.1 一次完整钓鱼的经济模型

以新手装备、长海县（免费入场）为例：

| 步骤 | 金币变化 |
|------|----------|
| 抛竿 | -1 鱼饵（~10 金币） |
| 装备耐久损耗 | ~2 金币修理费 |
| 钓到鱼（均价）| +45 金币 |
| **单次净赚** | **~33 金币** |

10 次钓鱼 → 净赚 ~330 金币 → 够买一个基础渔线升级

### 2.2 鱼价公式

```
卖鱼价格 = weight × fish.value × rarityMultiplier × (1 + goldBonus/100) × weightPremium
```

| 稀有度 | 系数 | 1kg 鱼示例(value=20) |
|--------|------|---------------------|
| ★ 常见 | ×1.0 | 20 金币 |
| ★★ 稀有 | ×1.8 | 36 金币 |
| ★★★ 史诗 | ×3.5 | 70 金币 |
| ★★★★ 神秘 | ×7.0 | 140 金币 |
| ★★★★★ 传说 | ×15.0 | 300 金币 |

### 2.3 消耗品定价

| 类别 | 价格范围 | 说明 |
|------|----------|------|
| 鱼饵(普通) | 5-20 金币 | 沙蚕、面团等，每次抛竿消耗 1 个 |
| 鱼饵(高级) | 20-80 金币 | 活虾、鱼块，提高稀有鱼概率 |
| 拟饵 | 30-100 金币 | 不消耗，可重复使用 |
| 窝料 | 50-300 金币 | 5-20 次使用，提高目标鱼概率 |

### 2.4 装备定价（按等级梯度）

| 等级段 | 渔竿 | 渔线 | 渔钩 | 卷线器 |
|--------|------|------|------|--------|
| 入门(1-5) | 500-1500 | 150-400 | 80-200 | 400-1000 |
| 进阶(5-15) | 1500-3000 | 400-800 | 200-400 | 1000-2000 |
| 高级(15-25) | 3000-5000 | 800-1200 | 400-600 | 2000-4200 |

### 2.5 修理费用

```
修理费 = 装备价格 × 损耗比例 × 0.3
```

| 场景 | 费用 |
|------|------|
| 渔竿 500g，损耗 50% | 75 金币 |
| 渔竿 3000g，损耗 20% | 180 金币 |
| 全部装备损耗 50% | ~200-500 金币 |

## 三、钻石经济

### 3.1 钻石定位

钻石是**便利货币**，不是 Pay-to-Win。所有钻石消费都有金币替代方案。

### 3.2 钻石获取

| 来源 | 数量 | 频率 |
|------|------|------|
| 签到 day3 | 5 | 每周 1 次 |
| 签到 day6 | 10 | 每周 1 次 |
| 签到 day7 | 30 | 每周 1 次 |
| 任务完成 | 2-5 | 每日 3-5 次 |
| 成就解锁 | 10-50 | 一次性 |
| 礼包码 | 不定 | 运营活动 |

**每周稳定收入**：签到 ~45 + 任务 ~50 = **~95 钻石/周**

### 3.3 钻石消费

| 用途 | 钻石 | 金币替代 |
|------|------|----------|
| 补签 | 20/次 | 看广告 |
| 鱼缸扩容 +1 | 50 | 5000 金币 |
| 立即完成鱼缸金币 | 5 | 等时间 |
| 改名卡 | 100 | 无替代 |
| 特殊皮肤 | 200-500 | 无替代 |

## 四、每日经济循环

```
登录 → 签到(100-2000g) → 领任务(3-5个)
  ↓
钓鱼 10-30 次 → 卖鱼(300-1500g) → 修理(0-200g)
  ↓
完成任务(200-500g + 钻石) → 鱼缸收取(50-200g)
  ↓
买新装备/窝料 或 攒钱
```

**新手每日预估收入**：500-1500 金币 + 10-20 钻石
**老手每日预估收入**：2000-5000 金币 + 15-30 钻石
