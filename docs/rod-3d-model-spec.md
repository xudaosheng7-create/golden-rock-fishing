# 游钓天下 — 3D 鱼竿模型需求

> 交付对象：AI 3D 模型生成工具（豆包等）
> 格式：GLB 2.0 / GLTF
> 目标平台：手机 H5（Three.js + React Three Fiber）

---

## 一、技术规格

| 参数 | 要求 |
|------|------|
| **格式** | `.glb`（首选）或 `.gltf` + `.bin` + 贴图 |
| **面数** | ≤ 3000 tris（手机性能） |
| **贴图** | 1 张合并贴图 1024×1024 或 512×512 PNG/JPEG |
| **材质** | PBR（BaseColor + Roughness + Metallic），单材质 |
| **单位** | 米（1 unit = 1m） |
| **朝向** | Y 轴向上为竿尖方向，Z 轴为前方 |
| **原点** | 握把底部为原点 (0, 0, 0) |

## 二、鱼竿造型

### 整体尺寸
- 总长：约 **1.8 米**
- 第一人称视角：握把在画面底部，竿尖指向前方偏上

### 分段结构（从下到上）

```
竿尖(红)
  │
  ├─ 竿梢节 — 细，深灰碳纤色，直径 2mm→5mm，长 60cm
  ├─ 第3节 — 渐粗，深灰碳纤色，直径 5mm→10mm，长 50cm
  ├─ 第2节 — 碳纤黑，直径 10mm→16mm，长 40cm
  ├─ 第1节 — 碳纤黑，直径 16mm→20mm，长 25cm
  │
  ├─ 卷线器座 — 金属银，有螺纹纹理
  │
  └─ 握把 — 软木色(浅棕)，直径 22mm，长 30cm
     └─ 尾堵 — 黑色橡胶/金属
```

### 导环（共 5 个）
- 从竿梢到竿身均匀分布
- 每个导环：金属支架 + 陶瓷内圈（银色/黑色）
- 尺寸从竿尖向握把递增（最小内径 3mm → 最大 12mm）

### 卷线器
- 位置：握把上方 5cm 处，竿身右侧
- 主体：金属灰银色，圆柱形
- 线杯：可转动，略深色
- 摇臂 + 摇把：竿身右侧伸出

## 三、材质要求

| 部位 | BaseColor | Roughness | Metallic |
|------|-----------|-----------|----------|
| 握把 | 浅棕 #c4956a | 0.7 | 0.0 |
| 竿身 | 深灰黑 #1a1a2e | 0.3 | 0.15 |
| 竿尖 | 荧光红 #ff3300 | 0.2 | 0.0 |
| 卷线器主体 | 银灰 #8899aa | 0.15 | 0.9 |
| 导环支架 | 银 #999999 | 0.2 | 0.9 |
| 导环内圈 | 陶瓷黑 #333333 | 0.1 | 0.05 |
| 尾堵 | 黑 #222222 | 0.5 | 0.3 |

## 四、动画要求（重要！）

模型需要支持以下变形/骨骼：

- **竿身弯曲**：从握把到竿尖，沿 Z 轴（前后方向）可弯曲 0-30°
  - 弯曲中心在竿身中部（约 60cm 处）
  - 竿尖弯曲幅度最大，握把不动
  - 类似鱼上钩后竿身被拉弯的效果

> 提示：如果有骨骼/蒙皮，设置 3-4 根骨骼沿竿身分布即可。
> 如果没有骨骼，至少把竿尖节做成独立 Mesh，代码里旋转它来模拟弯曲。

## 五、视角说明

此鱼竿用于**第一人称钓鱼游戏**，视角如下：
- 摄像机在玩家眼睛位置
- 鱼竿从画面中下方伸出，指向画面中央偏上
- 鱼竿底部（握把）靠近屏幕底边
- 鱼竿长度约占屏幕高度 60-70%

初期渲染位置参考（Three.js 世界坐标）：
```
position: [0.25, -3.0, -3.5]
rotation: [-0.35, 0.12, 0]
```
